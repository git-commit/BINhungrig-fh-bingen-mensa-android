BINhungrig - Mensa FH-Bingen

Speiseplan betrachten, Speisen bewerten und Fotos der Speisen aufnehmen.

Download this project as a .zip file Download this project as a tar.gz file

Hallo

Schau dir den Speiseplan an, bewerte die Gerichte und lade ein Foto der Speise hoch. Die Daten werden serverseitig aus dem PDF-Speiseplan des Studierendenweks Mainz extrahiert und von den mobilen Endgeräten angezeigt. Diese App gibt es auch für iOS.

Entstehung

Diese App entstand aus einem Projekt im Modul 'Mobile Computing' des Studiengangs Informatik der Fachhochschule Bingen.

Mitmachen

Gerne kannst du am Projekt teilnehmen! Fällt dir ein Fehler auf oder ein Feature ein, erstelle einen Issue auf GitHub. Oder $ git fork das Repo, entwickle mit und stelle Pull requests.
